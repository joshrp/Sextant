# Balancers

Balancers here a combination of in-game balancer and storage buildings.

They should be used to collate multiple outputs or inputs into a single value, so you can provide the resources for all the consumers in one place. 

Note that you don't **need** balancers to do this. You can connect multiple lines to the same input or output and the calculator will do the rest, balancers are a visual and organizational aid. 

### Balancing
Connecting multiple lines to a balancer input or output will let the calculator decide which path to take for your chosen [Scoring Method](help?topic=scoring).
